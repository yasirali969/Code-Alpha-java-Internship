
# 🎓 Student Grade Manager (GUI)

A simple Java Swing-based desktop application to manage student grades and generate basic performance statistics.

---

## 🚀 Features

* Add student name and grade using GUI

* View all student records in a table

* Automatically calculate:

  * Average score
    
  * Highest score

  * Lowest score

* Clean and modern Java Swing interface

* Input validation for correct grade entry

---

## 🛠️ Technologies Used

* Java

* Java Swing (GUI)

* AWT Event Handling

* Object-Oriented Programming (OOP)

---

## 🖥️ How It Works

1. Enter student name
2. Enter grade (0–100)
3. Click **Add Student Record**
4. Data is added to table automatically
5. Statistics update in real-time

---

## 📊 Output Includes

* Student Name List
* Grades Table
* Performance Summary Panel

---
